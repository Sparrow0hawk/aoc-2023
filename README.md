Solutions for [Advent of Code 2023](https://adventofcode.com/2023).

## Setup

You will need:
- Python 3.12

1. Create a virtual environment
    ```bash
    python -m venv --prompt . .venv
    ```
2. Activate the virtual environment
    ```bash
   . .venv/bin/activate 
   ```
3. Run the code for a day
    ```bash
   python -m day1
   ```
